
# Browser

Brower extensions and configurations

## Firefox

### Extension: Vimium

Key Mapping: ([Vimperator](https://addons.mozilla.org/en-US/firefox/addon/vimperator/) like keymapping)
```
unmap d
map d removeTab
unmap u
map u restoreTab
map <c-p> previousTab
map <c-n> nextTab
unmap yy
map y copyCurrentUrl

unmap J
unmap K
```
